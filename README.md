# layday/github-wow-addon-catalogue

Scrapes WoW addon information from Github repositories.

## requirements

* Python 3.10 or later
* Github developer token

## usage

1. Generate a GitHub token and write it to `github-token`
2. `./install.sh`
3. `./run.sh`
